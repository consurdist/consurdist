adopted by several oil and gas companies. 65,000+ lines solo. The system handles
depth-registered image compositing, vector annotation layers, real-time
collaborative sync, ML inference with retraining pipelines, and Jupyter
integration — built on a 25-slice Redux architecture managing thousands of
locally-sourced images and annotations simultaneously within strict enterprise
data security requirements. Full AWS infrastructure via CDK, Well-Architected,
FTR certified.

Before that: a stochastic trading signal pipeline on Twitter data, a face-swap
application, a self-directed research project called Hylomorphic Optics

phenomenology, AST analysis, and UI design), and enough range across unrelated
